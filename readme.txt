WP MusicMazaa makes it easier for you to embed MusicMazaa playlists in Wordpress using one simple tag.

== Installation ==

   1. Upload wp-musicmazaa.php to the /wp-content/plugins/wp-musicmazaa directory.
   2. Activate the plugin WP Musicmazaa through the ‘Plugins’ menu in WordPress.
   3. Go to ‘Options / WP Musicmazaa’ for options and more instructions.
